# 世界設定
## 時代
- 価値観・倫理観は現代基準
- ただしインターネット・ＳＮＳが登場しない
  - ネットで感じる価値観などは全て現実の描写に落とし込まれる
## 舞台
- 架空の地球、架空の日本
- 神戸市を連想させる山と海に挟まれた街『風見市』
- 国名は出さない
## 地域
- 震災は描写しないが、整理された街並みは「戦中の空襲による影響」とする
- 海沿いは整理された区画に真新しいビル群
- 山側には洋館・異人館街が並ぶ
## 歴史
- 開港後、外国人居留地が建築された
- 戦後も山間には当時の洋館街・異人館街が残っている
## 異能
### 嗜血病
- 年頃の少女が発症し、情緒不安定・突発的な攻撃衝動や同性に対する吸血衝動を抱える
- 診断された患者は、風見市の山間にある局立指定学校・月銀学園に入所する
