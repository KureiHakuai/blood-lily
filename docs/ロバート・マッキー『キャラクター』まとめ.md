# ロバート・マッキー『キャラクター』まとめ
## プロットVSキャラクター
### キャラクター
- 架空の存在
- 出来事を引き起こしたり、他者が引き起こした出来事に反応したりする
### プロット
- ストーリーにおける出来事の配置
### 出来事
- ストーリーにおいて何らかの価値ある変化（価値要素）を生む出来事
### 価値要素
- 二元的な人生経験
- プラスからマイナス、またはマイナスからプラスへ移り変わるもの
### 価値のある出来事の例
- 何かが起きてあるキャラが別のキャラに向ける感情が愛から憎しみに変わったら、その出来事は意味を持つ
- ある出来事によって貧乏だったキャラが裕福になったらその変化は意味を持つ
- そよ風が吹いて芝生の上に落ちたはが舞い上がることは、変化であっても価値のある出来事ではない
### 性格描写と実像
#### 性格描写
- すべての目に見える性質や表向きの行動
  - 年齢、性別、仕草、職業、住まい、服装、態度、人柄
#### 実像
- 目に見えないキャラクターの内なる性質
  - 心の奥底にある動機
  - 基本的な価値観
- 人生最大の重圧に晒されたときの選択や行動にあらわれる
### プロット主導とキャラクター主導のちがい
#### 因果関係
- プロット主導
  - 鍵となる転換点はキャラクターの手が届かない出来事になる
    - 疫病の蔓延、宇人の侵略、独裁者の宣戦布告など
- キャラクター主導
  - キャラクターの持つ願望、選択、行動が転換点となる
#### 独自性
#### 価値要素
- プロット主導
  - 主人公は世界に欠けているものをもたらそうとする
  - 例：戦争に対する平和や、富に対する貧困、病気に対する健康など
- キャラクター主導
  - 主人公は自分自身に欠けているものを手に入れようとする
  - 例：愛に対する憎しみ。信頼に対する不信。希望に対する絶望など
#### 深み
- プロット主導
  - キャラクターの潜在意識や矛盾はめったに描かれない
  - 例：ミッション・インポッシブルでイーサンが子供時代のトラウマに悩み計画を即断実行できなければ、スピード感に満ちていた物語は一気につまらなくなるだろう
- キャラクター主導
  - 心理面の矛盾をえがくことで物語を重層化する
  - ただしキャラクターが目前の敵対する力以上に複雑になることはない
  - 対立や葛藤によって顕在化することによって初めて、キャラクターの深みを知ることができる
#### 好奇心
- プロット主導
  - 内なる葛藤は排除される
  - 主要な人物たちは社会の中で二極化される
  - 例：アクション作品の主人公は不正を正して被害者を助ける。悪役は残虐にふるまう
- キャラクター主導
  - 物理的・社会的な対立や葛藤は最小限
  - キャラクターの内なる闘いや他キャラとの個人的な争いに焦点を当てる
#### 運命と自由
- プロット主導
  - 契機事件においてキャラクターはプラスかマイナスどちらかの運命を追い求める
- キャラクター主導
  - 未来を決めるのは、複雑なキャラクターの内面にある矛盾した力
