# 新人賞投稿用百合ライトノベル
## 想定概要
- 百合またはそれに準ずるものを対象に含めているライトノベル系レーベル新人賞への応募
- 文字数は8万字以上を想定
## ジャンル
- 百合
- ライトノベル
## ログライン
- お嬢さまに憧れて上京した田舎娘が入学したのは、吸血鬼たちの女学校だった
- 憧れを抱く女の子が、夢を叶える物語
- 吸血鬼
- 学園異能バトル
## プロット
### 第一話
- お嬢さまに憧れた主人公・櫻木あみかは、名門女学校の入学試験を受ける
- おっちょこちょいなあみかは何をやってもダメ。うまくいかない（おそざきのレオ）
- 吸血鬼の魔の手が伸びる。土壇場で主人公は勇気を見せる
- 勇気を以て吸血鬼を撃退する。その功績から主人公は合格する

## プロット

- ジャンルの提示

    - 設定までには踏み込まない
  - 百合（主人公CPの関係性）
    - 幼馴染といった設定ではなく、会話からCP魅力を滲ませる
  - 吸血鬼
    - 吸血シーン
- 主人公の目的
  - 吸血鬼だとバレてはいけない（秘密という枷）
