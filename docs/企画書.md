# 新人賞投稿用百合ライトノベル
## 想定概要
- 百合またはそれに準ずるものを対象に含めているライトノベル系レーベル新人賞への応募
- 文字数は8万字以上を想定

- ライトノベル
- 女児アニメ
## テーマ
- 人間の魂の輝き
  - デミカツ終了でメエさんが真摯に朝配信をしてくれたこと
  - 絶望的な状況下でフアンがそれでも応援しようとしたこと
  - 最後までやりきろうとすること
  - やらなくてもいいのに、最後まで誠実であるとすること
  - お別れだけど、この人ならこの先どんな場所でも大丈夫だし、大丈夫であってほしいと思うこと
  - 追い詰められた状況下、もうおしまいという時、その土壇場で意味を為す人間の高潔さ
  - 高潔さは日常において簡単に裏切られ意味を為さないが、高潔さが価値を発揮するのは全てが追い詰められた土壇場である
  - 配信がなくなること＝吸血鬼という存在が世界から消えること
